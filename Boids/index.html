Ensemble de Mendelbrot

L'Ensemble de Mandelbrot est l'ensemble de tous les nombres complexes c pour lesquels la fonction
fc(z) = z2 + c ne diverge pas lorsqu'elle est itérée à partir de z = 0, c'est-à-dire que la séquence fc(0), fc(fc(0)), fc(fc(fc(0))), ... reste bornée.

Règles pour la production visuelles

L'ensemble de Mandelbrot est souvent visualisé en colorant les points du plan complexe (représentant différentes valeurs de c) en fonction du nombre d'itérations nécessaires pour que zn+1 dépasse la valeur limite. Les points qui ne dépassent pas la limite après un grand nombre d'itérations sont généralement considérés comme faisant partie de l'ensemble de Mandelbrot.

Cette structure complexe et fractale génère des motifs incroyablement détaillés et intéressants lorsqu'elle est visualisée graphiquement, ce qui en fait l'un des exemples les plus célèbres de mathématiques visuelles et de fractales.

  • On commence par convertir les coordonnées de chaque pixel en une fonction complexe.

  • On Commence avec z0 = 0.

  • Pour chaque itération n, on calcule zn+1 = zn2 + c.

  • Si le module de zn+1 (c'est-à-dire sa distance par rapport à l'origine dans le plan complexe) devient supérieur à une certaine valeur limite (par exemple, 22), alors c n'appartient pas à l'ensemble de Mandelbrot.

  • Si le module de zn+1 reste borné (ne devient jamais trop grand), alors c appartient à l'ensemble de Mandelbrot.

Des exemples de constante :

  • c = 0.285 + 0i
  • c = 0.285 + 0.01i
  • c = 0.45 + 0.1428i
  • c = −0.70176 − 0.3842i
  • c = c = −0.835 − 0.2321i
  • c = −0.8 + 0.156i
  • c = −0.7269 + 0.1889i
