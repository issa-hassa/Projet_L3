Tri par Insertion

Le tri par insertion est un algorithme de tri efficace pour de petites quantités de données. Il fonctionne en construisant progressivement une liste triée à partir d'une liste non triée.

Fonctionnement

Le tri par insertion fonctionne comme suit :

  1. Début avec le premier élément de la liste et considérez-le comme une liste triée.
  2. Prenez le prochain élément non trié et insérez-le dans la liste triée à la bonne position.
  3. Répétez l'étape 2 jusqu'à ce que tous les éléments soient triés.

Exemple

Supposons que nous voulions trier le tableau [7, 2, 4, 1, 5]. Le tri par insertion fonctionnerait comme suit :

  • Étape 1 : La liste triée est [7].
  • Étape 2 : Insérer 2 dans la position correcte, la liste triée devient [2, 7].
  • Étape 3 : Insérer 4 dans la position correcte, la liste triée devient [2, 4, 7].
  • Étape 4 : Insérer 1 dans la position correcte, la liste triée devient [1, 2, 4, 7].
  • Étape 5 : Insérer 5 dans la position correcte, la liste triée devient [1, 2, 4, 5, 7].
  • Résultat : Le tableau trié est [1, 2, 4, 5, 7].

Avantages et Limitations

Le tri par insertion est adapté pour de petites quantités de données ou lorsque la liste est déjà partiellement triée. Cependant, il n'est pas très efficace pour de grandes listes car sa complexité temporelle est de l'ordre de O(n^2).
